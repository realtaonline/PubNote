1. Render a PubMed instance, both input and output - PubNoteRender.xsl

Filename: PubNoteRender.xsl

Include statement:

  • PubMed In and Out DTD element names with English expansions - PubNote-xlate-en.xsl

BSD-3 License

This stylesheet creates an XSL-FO instance suitable for publishing to PDF and for downstream processes to render to HTML and DOCX.

1.1. Invocation parameters, globals, and input file

The input file is assumed to be a Pubmed document, either input or output.

pn:pageWidth as="xsd:string" select="'210mm'" (xsl:variable)

Page dimension to be overridden when needed

pn:pageHeight as="xsd:string" select="'297mm'" (xsl:variable)

Page dimension to be overridden when needed

pn:elementNames match="element" use="@name" (xsl:key)

Quick index into the element name translation list

1.2. Main logic

match="/" (xsl:template)

Establish the page geometry and flow

match="*" (xsl:template)

The handling of every element is the same, save for title formatting of the document element.
